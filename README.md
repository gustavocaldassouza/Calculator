# Calculator (SwiftUI)

A colorful, modern calculator app built with SwiftUI.

Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal input, percent, plus/minus toggle, and clear (AC)
- Colorful UI with gradient background and styled buttons

How to run

1. Open `Calculator.xcodeproj` in Xcode.
2. Select a simulator or device.
3. Build and run (⌘R).

Files

- `CalculatorApp.swift` — App entry point
- `ContentView.swift` — Main SwiftUI view with calculator UI and logic

Notes

- The UI uses SwiftUI and is designed for modern iOS/macOS targets that support SwiftUI.
- If you run into layout issues on different screen sizes, adjust the button sizing logic in `ContentView.swift`.
